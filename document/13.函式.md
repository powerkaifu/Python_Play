# 定義函式

- 無參數函式

```python
def greet():
    print("Hello")
```

- 有參數函式

```python
def greet(name):
    print("Hello", name)
```

- 有回傳值的函式

**沒有回傳值的函式會回傳 None**。

```python
def add(a, b):
    return a + b
```

- 預設參數

```python
def greet(name="world"):
    print("Hello", name)
```

# 傳值與傳參考

- 傳值

適用於不可變物件，如 int、str、tuple。

```python
def add(a):
    a += 1

x = 10
add(x)
print(x) # 輸出: 10，x 並未被改變
```

- 傳參考

適用於可變物件，如 list、dict、set。

```python
def add(a):
    a.append(1)

x = [10]
add(x)
print(x) # 輸出: [10, 1]，x 被改變
```

# 回傳值可以多個

```python
def multi_return():
    return 1, 2

a, b = multi_return()

print(a, b) # 輸出: 1 2
```

# lambda 函式

lambda 函式是一種匿名函式，可以用來簡化程式碼。

- 不能有區塊結構，只能處理單一運算式，並且會回傳運算式的結果
- 不需要使用 return

```python
add = lambda a, b: a + b
print(add(1, 2)) # 輸出: 3
```
