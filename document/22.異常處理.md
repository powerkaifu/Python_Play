# 範例

```py
try:
  word = 'Python'
  word[100] = 'H' # TypeError，字串不支持賦值
  a = word[100] # IndexError，索引超出範圍
except IndexError as e:
  print(f"索引超出範圍: {e}")
except TypeError as e:
  print(f"不支持賦值: {e}")
except Exception as e:
  print(f"捕捉到其他錯誤: {e}")
  print(f"錯誤類型: {type(e).__name__}")
```
