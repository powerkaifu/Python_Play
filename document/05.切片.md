# 切片

切片有三個參數：起始索引、結束索引、步長(step)。
起始索引是包含的，結束索引是不包含的。

```python
import numpy as np

# 切片適合有序的資料型態，例如字串、元組、列表等，字典不支持切片操作，因為字典是無序的。
# 切片的語法是 [start:stop:step]，start 是起始索引，stop 是結束索引，step 是步長

# 字串切片
str = "Hello, World!"
print(str[1:5])  # 輸出: ello

# Tuple 切片

tuple = (1, 2, 3, 4, 5)
print(tuple[1:3])  # 輸出: (2, 3)

# List 切片

list = [1, 2, 3, 4, 5]
print(list[1:3])  # 輸出: [2, 3]

# NumPy 陣列切片
print("-" * 30)
print("NumPy 陣列切片")
arr = np.array([1, 2, 3, 4, 5])
print(arr[1:3])  # 輸出: [2 3]
```

```python
words = "Hello, World!"
print(words[:]) # 輸出: Hello, World!
print(words[7:12])  # 輸出: World
print(words[-6:])  # 輸出: World!
print(words[::-1])  # 輸出: !dlroW ,olleH (反轉字串)
```
