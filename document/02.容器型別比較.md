# 容器型別比較

| 容器型別         | list(串列) | tuple(元組) | set(集合) | dict(字典)         |
| ---------------- | ---------- | ----------- | --------- | ------------------ |
| 符號             | []         | ()          | {}        | {}                 |
| 有無順序         | 有         | 有          | 無        | 有(3.7 之後有順序) |
| 可否改變內容     | 可以       | 不可以      | 可以      | 可以               |
| 可否重複         | 可以       | 可以        | 不可以    | 鍵不可以，值可以   |
| 可否放入可變物件 | 可以       | 可以        | 不可以    | 鍵不可以，值可以   |

# 可以用比較運算子比較

大部分物件都可以用比較運算子比較，
字串、數字、集合、列表、元組、字典等。

```py
# list
list1 = [1, 2, 3]
list2 = [3, 2, 1]
print(list1 == list2)  # False


# set
set1 = {1, 2, 3}
set2 = {3, 2, 1}
print(set1 == set2)  # True
```
